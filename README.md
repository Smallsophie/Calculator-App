# Calculator Web Application

A responsive calculator built using vanilla JavaScript, designed to handle real-time user input and dynamic calculations through clean UI interaction.

This project focuses on implementing core arithmetic logic, managing application state, and building a smooth, intuitive interface from scratch.

## Features

- Real-time arithmetic calculations
- Responsive dark-themed interface
- Clear and reset functionality
- Input validation handling
- Smooth button interaction feedback
- Mobile-friendly layout

---

## Technical Highlights

- DOM manipulation for dynamic UI updates
- Event-driven state management
- Expression evaluation handling
- Input control to prevent invalid operations
- Separation of structure (HTML), styling (CSS), and logic (JavaScript)
- Responsive layout using Flexbox

- CSS3
